# ng2-fileupload

### To install the server & client
```
$ npm install
$ cd public/testupload/
$ npm install
$ cd ../..
```

### To start the server:

```
npm start
```

Open browser to localhost:3000 and you will see the 'Welcome to Express' message


### To start the client:

```
cd public/testupload
ng serve
```

Open localhost:4200

The files will be uploaded in the directory called 'uploads'
